## Project Description
This is an updated, interactive personal portfolio website built for Assignment 2. It builds upon the HTML/CSS foundation of Assignment 1 by introducing JavaScript-driven interactivity, data handling, and animations to enhance the user experience.

## New Features (Assignment 2)
- **Dynamic Content & Data Handling:** A Dark/Light mode toggle that saves the user's preference using browser `localStorage`.
- **Animations:** Smooth CSS transitions for theme switching and interactive hover effects on project cards and buttons.
- **Error Handling & User Feedback:** Client-side JavaScript form validation that prevents empty submissions and displays dynamic success or error messages.
